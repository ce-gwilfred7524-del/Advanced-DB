## 🚀 How to Run

### Option 1 — Open locally
Just open `index.html` in any browser. No server needed.

### Option 2 — VS Code Live Server
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**

### Option 3 — Host on GitHub Pages
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Deploy from `main` branch
3. Share the link with classmates!

## ✨ Features

- **AI Explain** — Click on any question for a live AI explanation powered by Claude
- **Show Answer** — Instantly reveal the correct answer
- **Save Answer** — Type and save your own answer
- **Progress Tracking** — Tracks which questions you've answered (saved in browser)
- **Search** — Search across all questions in a section
- **Keyboard Shortcuts** — Press 1-5 to switch sections
- **MCQ Mode** — Click options and see immediate feedback + explanation
- **Reset** — Clear all progress to start fresh
